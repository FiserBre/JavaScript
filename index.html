Výuka JavaScriptu (ES6+)

Vítejte v interaktivní učebnici. JavaScript je mozkem webových stránek. Zatímco HTML dává strukturu a CSS vzhled, JS dává život.

2. Proměnné

V moderním JS používáme primárně let a const. Zapomeňte na staré var.

javascript 
let jmeno = "Petr"; // Hodnota, která se může měnit
const pi = 3.14;    // Konstanta, nelze přepsat

jmeno = "Jan";      // OK
// pi = 3.15;       // CHYBA!

3. Datové typy

  • String: "Text"
  • Number: 42 nebo 3.14
  • Boolean: true / false
  • Array: [1, 2, 3]
  • Object: { klic: "hodnota" }

5. Podmínky

Rozhodování v kódu pomocí if, else if a else.

let vek = 20;
if (vek >= 18) {
    console.log("Můžeš vstoupit");
} else {
    console.log("Vstup zakázán");
}

7. Funkce

Funkce jsou bloky kódu, které můžete volat opakovaně.

// Klasická funkce
function pozdrav(jmeno) {
    return "Ahoj " + jmeno;
}

// Moderní Arrow Function
const vynasob = (a, b) => a * b;

console.log(vynasob(5, 5)); // 25

11. Asynchronní JavaScript

Důležité pro načítání dat z internetu (API) bez zaseknutí stránky.

async function nactiData() {
    try {
        let response = await fetch('https://api.example.com/data');
        let data = await response.json();
        console.log(data);
    } catch (error) {
        console.error("Něco se nepovedlo");
    }
}
