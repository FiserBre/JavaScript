JavaScript: Kompletní Průvodce

JavaScript je skriptovací jazyk, který umožňuje vytvářet dynamicky se aktualizující obsah, ovládat multimédia, animovat obrázky a mnoho dalšího.

1. Proměnné a Scope

V moderním JavaScriptu zapomeňte na var. Používáme:

  • let - pro proměnné, které budeme měnit.
  • const - pro hodnoty, které zůstávají konstantní.
const dph = 0.21;
let cenaBezDane = 100;
let celkem = cenaBezDane * (1 + dph);

console.log(celkem); // 121

2. Datové typy

JavaScript je dynamicky typovaný jazyk. To znamená, že nemusíte specifikovat typ proměnné při jejím vytváření.

let text = "String";    // Text
let cislo = 42;          // Number
let pravda = true;       // Boolean
let prazdno = null;      // Null (záměrně prázdné)
let nedefinovano;        // Undefined

3. Funkce a Arrow Functions

Funkce jsou základní stavební kameny. Moderní zápis "Arrow function" je kratší a elegantnější.

// Klasický zápis
function pozdrav(jmeno) {
    return `Ahoj ${jmeno}!`;
}

// Arrow function (moderní)
const pozdravModerni = (jmeno) => `Ahoj ${jmeno}!`;

4. Pole a moderní metody

Práce s poli je v JS extrémně silná díky metodám jako map, filter a forEach.

const cisla = [1, 2, 3, 4, 5];

// Vytvoří nové pole, kde jsou všechna čísla dvojnásobná
const dvojnasobek = cisla.map(n => n * 2); 

// Vyfiltruje pouze sudá čísla
const suda = cisla.filter(n => n % 2 === 0);

6. Asynchronní JS (Fetch API)

Většina moderních aplikací komunikuje se serverem pomocí fetch a async/await.

async function ziskejData() {
    try {
        const response = await fetch('https://jsonplaceholder.typicode.com/users/1');
        const user = await response.json();
        console.log(`Uživatel: ${user.name}`);
    } catch (err) {
        console.error("Chyba při načítání:", err);
    }
}
