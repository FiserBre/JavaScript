Základy JavaScriptu

Vše, co potřebuješ vědět, než začneš stavět aplikace.

1. Kde se píše kód?

JavaScript vkládáme do HTML pomocí tagu <script>, nebo ho připojujeme jako externí soubor:

<script src="script.js"></script>

2. Proměnné (Krabice na data)

Proměnné slouží k ukládání hodnot. Dnes používáme výhradně let a const.

let Pro hodnoty, které se budou měnit (např. skóre v aplikaci).
const Pro hodnoty, které jsou pevné (např. ID uživatele). 
let body = 10;
body = 20; // Funguje

const jmeno = "Petr";
// jmeno = "Jan"; // CHYBA!

3. Základní datové typy

JavaScript automaticky pozná, o jaký typ dat jde:

  • String: Text v uvozovkách "Ahoj"
  • Number: Čísla 42 nebo 3.14
  • Boolean: Logická hodnota true nebo false

4. První interakce

Zkuste tyto příkazy pro výpis dat:

console.log("Zpráva do konzole"); // Pro vývojáře
alert("Vyskakovací okno");       // Pro uživatele
prompt("Jak se jmenuješ?");      // Vstup od uživatele

⚡ Mini Úkol

Zkuste si v konzoli prohlížeče (F12) napsat tento kód:

const jmeno = prompt("Zadej své jméno:");
console.log("Vítej ve světě JS, " + jmeno + "!");
